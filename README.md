# Нейросети в промышленном туризме

MVP для туроператора: пользователь вводит параметры группы, система подбирает предприятия, транспорт, питание и размещение, формирует программу по минутам, стоимость и публичную ссылку для клиента.

## Запуск

```bash
npm start
```

Приложение работает на:

```text
http://localhost:4173
```

В этом рабочем сеансе сервер уже запущен.

## Что реализовано

- Сборка тура по параметрам: город, тип группы, количество человек, длительность 3/5/7 часов, цель визита, интересы, ночёвка.
- Подбор 1-3 предприятий из тестовой базы.
- Подбор транспорта, питания и размещения.
- Расчёт стоимости на группу и на человека кодом, не LLM.
- Программа тура по минутам.
- Ручная корректировка предприятий, питания, транспорта и размещения.
- Публичная ссылка клиента `/proposal/:code`.
- Бронирование и mock-оплата.
- Mock email-уведомление при бронировании.
- Админ-раздел справочников.
- Настройки LLM через интерфейс: URL, ключ, модель, системный промпт.

## LLM-интеграция

Сделано по тому же принципу, что в первом примере:

- backend хранит локальные настройки в `server/data/ai-cred.json`;
- frontend не видит API-ключ;
- сервер отправляет запрос в OpenAI-compatible endpoint `/v1/chat/completions`;
- модель получает только параметры группы и отфильтрованных кандидатов из базы;
- если ключ не указан или провайдер недоступен, работает fallback-подбор правилами.

Важно: `server/data/ai-cred.json` добавлен в `.gitignore` и не должен попадать в GitHub. Для репозитория есть только безопасный шаблон `server/data/ai-cred.example.json` без ключа. Ключ вводится локально через UI или вручную в `ai-cred.json`.

По умолчанию:

```json
{
  "AI_API_URL": "https://api.openai.com/v1/chat/completions",
  "AI_API_MODEL": "gpt-3.5-turbo"
}
```

Модель можно заменить в UI без переписывания кода.

## Важное по логике

LLM не считает финальную цену и не решает жёсткие ограничения. Код проверяет город, вместимость, доступность объектов, длительность и стоимость. LLM используется для мягкого ранжирования, объяснения выбора и интерпретации интересов группы.
